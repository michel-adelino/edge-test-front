# Edge Server Dashboard

A simple, premium dashboard to manage Guests and Orders for the Edge Server project.

## Features
- **Guest Management**: View, Add, Edit, Delete guests.
- **Order Management**: View, Add, Edit, Delete orders with item support.
- **Rich Aesthetics**: Glassmorphism design with dark mode.

## Setup

1. **Backend**:
   - Navigate to `../edge-server-test`
   - Run `npm install`
   - Run `npm start` (or `node index.js`)
   - Server runs on `http://localhost:5000`

2. **Frontend**:
   - Navigate to this directory (`edge-test-front`)
   - Run `npm install`
   - Run `npm run dev`
   - Open browser at `http://localhost:5173`

## Tech Stack
- React + Vite + TypeScript
- Vanilla CSS (Variables, Glassmorphism)
- Axios
- React Router DOM
